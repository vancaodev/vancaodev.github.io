Giới Thiệu

Lập trình viên C# (.NET) tập trung vào giải pháp thực tế và code sạch, dễ bảo trì.

Tôi là Lập Trình Viên C# (.NET) chuyên xây dựng ứng dụng quản lý và phần mềm desktop với Windows Forms và WPF. Tôi cũng có kinh nghiệm thực tế với PHP, HTML, CSS, JavaScript, và Bootstrap để tạo giao diện web hiện đại khi cần thiết.

Hiện tại tôi đang phát triển các kênh YouTube và TikTok dành riêng cho các hướng dẫn chi tiết, từng bước về cách hoàn thành các dự án/đồ án C# — nội dung dài trên YouTube và các tips ngắn gọn trên TikTok. Mục tiêu của tôi là giúp sinh viên và lập trình viên hoàn thành dự án với sự rõ ràng và tự tin.

Tôi cung cấp dịch vụ làm đồ án theo yêu cầu với code sạch, tài liệu đầy đủ, hướng dẫn cài đặt, và hỗ trợ chỉnh sửa cho đến khi nộp bài cuối cùng.

Kỹ Năng Chính

Các công nghệ tôi sử dụng để xây dựng phần mềm nhanh, ổn định và thân thiện với người dùng.

C# / .NET

Windows Forms, WPF, nguyên tắc SOLID, kiến trúc MVVM, giải pháp code sạch.

PHP & Web Frontend

PHP, HTML, CSS, JavaScript, Bootstrap cho giao diện web responsive và hiện đại.

Cơ Sở Dữ Liệu

MySQL & SQL Server: thiết kế, truy vấn, tối ưu hiệu năng, đảm bảo tính toàn vẹn dữ liệu.
